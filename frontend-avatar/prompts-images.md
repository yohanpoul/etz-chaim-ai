# Prompts de generation d'images pour les 20 avatars

> Style global : 3D character portrait, Pixar-quality rendering, noble archetypal figures.
> Fond : pure black. Eclairage : warm amber rim light, dramatic.
> Camera : bust portrait, 3/4 angle sauf Nukva (face).
> Coherence : TOUS les personnages doivent sembler venir du meme univers visuel.

## Notes de coherence

1. **Palette** : tous les personnages baignent dans l'amber (#ffb000) sauf :
   - Raphael : touche de vert dans sa lumiere
   - Samael : tons plus froids
   - L'Ombre : rouge
   - La Lumiere : blanc-or
2. **Proportions** : memes proportions Pixar pour tous (tete legerement plus grande)
3. **Fond** : toujours noir pur (#0a0a0a)
4. **Vetements** : coherents entre eux (meme univers — ni medieval pur, ni moderne, intemporel)
5. **Regard** : tous en 3/4 sauf Nukva (face) et Beinoni (face)
6. **Age** : Arikh (ancien), Abba/Imma (matures), Zeir Anpin (jeune), Archanges (ageless)
