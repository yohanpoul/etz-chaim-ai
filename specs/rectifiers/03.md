# Rectifier 03 — Insight Grounding

## Failure pattern

Insight emitted with no traceable provenance to the source corpus or to
prior memory entries. Indicates ungrounded generation (hallucination
proxy).

```python
{
    "rectifier_id": "r03",
    "pattern": "ungrounded_insight",
    "metrics": {
        "ungrounded_count_24h": int,
        "ratio_ungrounded": float,
    },
}
```

## Actions

- `observe` : log only.
- `suggest` : emit `r03_action_proposed` recommending re-grounding via
  the Insight faculty's `seek_provenance(insight_id)`.
- `act` : invoke `seek_provenance`; if no source found within 3 attempts,
  flag the insight `quarantine` (not deleted, just demoted).

## Invariants

1. Never modifies the insight content; only its `quarantine` flag.
2. Quarantine is reversible : a future grounding pass can lift it.
3. Bounded retries : max 3 grounding attempts per insight per cycle.

## Tests

- Insight with no provenance → deviation detected.
- Insight with provenance → no deviation.
- `act` flags insight with no source after 3 retries.
- Quarantine reversible via `unquarantine(insight_id)`.
