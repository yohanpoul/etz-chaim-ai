# Rectifier 09 — Cross-Coupling Overload

## Failure pattern

Two configurations request boosts on the same faculty within a short
window, causing cumulative trace coefficient to overshoot the 0.3 plateau
ceiling.

```python
{
    "rectifier_id": "r09",
    "pattern": "coupling_overload",
    "metrics": {
        "concurrent_boosts": int,
        "overshoot_amount": float,
        "affected_faculty": str,
    },
}
```

## Actions

- `observe` : log only.
- `suggest` : emit `r09_action_proposed` recommending boost coalescing
  (one larger boost replacing two simultaneous ones).
- `act` : apply boost coalescing; clip the result at plateau ceiling.

## Invariants

1. Trace coefficient never exceeds plateau (0.3) post-rectification.
2. Coalescing preserves total boost intent (sum of source magnitudes
   capped at plateau).
3. Originating configurations notified of the coalescing.

## Tests

- Two boosts overshooting → deviation detected.
- Two boosts within ceiling → no deviation.
- `act` clips at plateau.
- Both source configurations receive notification.
