# Rectifier 04 — Goal Drift

## Failure pattern

Active intentions diverging from MISSION.md north star. Measured by
embedding cosine distance between intention text and mission claim,
averaged over the active intention set.

```python
{
    "rectifier_id": "r04",
    "pattern": "goal_drift",
    "metrics": {
        "active_intentions": int,
        "avg_distance_to_mission": float,
        "outlier_intentions": list[str],
    },
}
```

## Actions

- `observe` : log only.
- `suggest` : emit `r04_action_proposed` listing intentions with
  distance > threshold (default 0.4 on cosine [0,1]).
- `act` : park outlier intentions with status `pending_alignment` and
  emit a request to the Tension faculty for review.

## Invariants

1. Never deletes intentions; only flags status.
2. Mission file change resets the distance baseline (no false drift).
3. Threshold is configurable per project.

## Tests

- All intentions on-mission → no deviation.
- One intention high distance → flagged in `outlier_intentions`.
- `act` parks the right intentions; mission text change clears flags.
- Threshold respects config override.
