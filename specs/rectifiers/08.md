# Rectifier 08 — Exploration Starvation

implemented in `probes/`; this spec captures the contract for
public neutral packaging.

## Actions

Existing implementation supports `observe` / `suggest` / `act` ; in `act`
mode it adjusts `novelty_threshold` (-0.1) and `breadth` (+5) on the
Exploration faculty.

## Invariants

1. Boost limits respect Exploration's calibration ceiling.
2. Reversible via `restore_calibration()`.
3. No-op when Exploration is intentionally paused (e.g. config flag).
