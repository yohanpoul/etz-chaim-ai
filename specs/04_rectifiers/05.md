# Rectifier 05 — Tension Freeze

## Failure pattern

Open dissensus tickets older than N days with no progress events
(no new evidence, no votes, no synthesis attempt). Stuck conflicts
in the Tension faculty.

```python
{
    "rectifier_id": "r05",
    "pattern": "tension_freeze",
    "metrics": {
        "open_tickets": int,
        "frozen_over_7d": int,
        "oldest_age_days": float,
    },
}
```

## Actions

- `observe` : log only.
- `suggest` : emit `r05_action_proposed` recommending forced advance
  (request fresh evidence or auto-park).
- `act` : for tickets frozen > 14 days without evidence : park status
  with reason `insufficient_evidence`. For tickets with evidence but
  no synthesis : trigger a synthesis attempt via the Synthesis Bridge.

## Invariants

1. Never auto-resolves a tension; only parks or routes.
2. Parked tickets remain queryable and re-openable.
3. Synthesis attempt does not write the verdict; only proposes one.

## Tests

- Frozen ticket > 14d without evidence → parked.
- Frozen ticket with evidence → synthesis attempt triggered.
- Active ticket (evidence < 14d old) → no action.
- Reversible : park status can be lifted.
