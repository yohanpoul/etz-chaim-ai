# Rectifier 06 — Self-Model Overconfidence

## Failure pattern

Self-model predictions diverging from outcomes : Brier score over the
last N predictions exceeds threshold, indicating overconfident
self-assessment.

```python
{
    "rectifier_id": "r06",
    "pattern": "selfmodel_overconfidence",
    "metrics": {
        "predictions_24h": int,
        "brier_score": float,
        "calibration_curve_deviation": float,
    },
}
```

## Actions

- `observe` : log only.
- `suggest` : emit `r06_action_proposed` recommending recalibration.
- `act` : apply temperature scaling on the self-model's confidence head;
  log calibration delta for post-mortem review.

## Invariants

1. Never modifies the underlying model weights; only output calibration.
2. Calibration adjustment is bounded (`|temperature - 1.0| < 0.5`).
3. Reversible : reset_calibration() restores temperature 1.0.

## Tests

- Brier > threshold → deviation detected.
- Brier ≤ threshold → no deviation.
- `act` adjusts temperature within bounds.
- Reset restores baseline.
