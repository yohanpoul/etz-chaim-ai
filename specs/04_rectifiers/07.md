# Rectifier 07 — Faculty Starvation

## Failure pattern

A non-exploration faculty has produced no output for > 7 days while the
agent is active. Indicates routing bias starving a capability.

```python
{
    "rectifier_id": "r07",
    "pattern": "faculty_starvation",
    "metrics": {
        "starved_faculties": list[str],
        "max_silence_days": float,
    },
}
```

## Actions

- `observe` : log only.
- `suggest` : emit `r07_action_proposed` recommending dispatch boost
  for starved faculties.
- `act` : adjust the meta-orchestrator's routing weights to nudge work
  toward starved faculties for the next 24h. Bounded boost (max +20%).

## Invariants

1. Boost is temporary (auto-revert after 24h).
2. Boost magnitude bounded; cumulative boosts capped.
3. Never silences other faculties below their minimum activity floor.

## Tests

- Faculty silent 8 days → flagged.
- Faculty silent < 7 days → not flagged.
- `act` boosts within bounds; auto-reverts after 24h.
- Other faculties stay above minimum activity.
